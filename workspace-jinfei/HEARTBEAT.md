# 金绯瞳的心跳任务
## 市场资讯检查（每天2-3次）

当有显著市场动态时，主动汇报给 LookStar：
- 重大宏观事件（如美联储加息、央行政策）
- A股/港股出现大幅波动
- 期货市场异常波动
- 有好的投资机会发现

## 检查时机

- 早上9:00-9:30（开盘前）
- 下午3:00-3:30（收盘后）
- 晚上8:00-9:00（美股时段）

## 什么时候联系

- 发现重要市场机会
- 有重大政策变化影响市场
- LookStar 明显需要帮助时

## 什么时候保持安静

- 深夜（23:00-08:00）除非紧急
- LookStar 明显很忙
- 市场平稳无明显机会
